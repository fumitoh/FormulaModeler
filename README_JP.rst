FMLの参照
--------
チュートリアル(Tutorial_0_4_0.xlsm)または保険数理モデル(FML_Actuarial_Model_0_4_0.xlsm)を実行するには、VBAコードから、FML.xlamを参照します。

FML.xlamを参照するには、VBA画面のメニューから [ツール] → [参照設定]を選択し、FLM.xlamが参照可能なライブラリファイルの一覧で、チェックボックスがチェックされていることを確認します。

チェックボッククスがチェックされていなかったり、チェックされていても参照不可となっている場合は、[参照]ボタンからFML.xlamを選択します。

ヘルプファイルに関する注意
------------------
ヘルプファイルをダブルクリックして開く場合、セキュリティの警告が出る場合があります。
ヘルプファイルの内容がうまく表示されない場合、"セキュリティの警告"のダイアログボックスに表示されているチェックボックス
"このファイルを開く前に常に警告する"のチェックボックスを外して下さい。
